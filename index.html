Mis Habilidades

Algunas de las tecnologías y habilidades que poseo

HTML5

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas webs. Sirve para indicar como va ordenado el contenido de una página web

CSS3

CSS (hojas de estilo en cascada) es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado (por ejemplo, HTML)

JavaScript

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web dinámicas. Nos permite realizar efectos, animaciones, acciones, etc.

ReactJs

React es una biblioteca Javascript diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

Bootstrap

Bootstrap es un framework CSS y Javascript diseñado para la creación de interfaces limpias y con un diseño responsive. Ofrece un amplio abanico de herramientas y funciones

Jquery

jQuery es una biblioteca multiplataforma de JS que permite manipular el DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web

NodeJs

Node.js es un entorno JavaScript de lado de servidor que utiliza un modelo asíncrono y dirigido por eventos. Tiene una arquitectura orientada a eventos y basado en el motor V8 de Google.

MySql

MySQL es un sistema de administración de bases de datos (DBMS) para bases de datos relacionales. Es una aplicación que permite gestionar archivos llamados de bases de datos.

Responsive Design

Responsive Design es una de las técnicas más utilizadas en la actualidad, permitiéndonos hacer que nuestros sitios web puedan adaptarse a cualquier dispositivo móvil o desktop.
